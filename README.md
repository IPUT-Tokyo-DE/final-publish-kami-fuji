# Jump Clime!!

## ゲームの内容
下から迫ってくるマグマをよけ、体力を保持しながらゴールを目指すゲームです。

![画面イメージ](docs/images/game_image01.png)

## 操作説明
例
WASDキー ：前後左右の移動
Spaceキー：ジャンプする。ジャンプの高さは自身のスピードに由来します。
Shiftキー：ダッシュします。ダッシュしている際はジャンプ力も高くなります
